# Sopia — Project Guidelines

## Branding

- **Name:** Sopia
- **Domain:** sopia.xyz
- **Tagline:** "Senior expertise where it matters. AI guidance everywhere else."
- **Subheadline:** "Sopia walks your team through every procedure step by step — so your best people stop repeating themselves and start doing the work that actually moves the needle."

## UI Direction

- **Fonts:** Geist Sans (primary) + Geist Mono (code/mono)
- **Primary color:** Teal — conveys trust, calm, guidance
- **Component library:** shadcn/ui (New York style)
- **Dark mode:** Yes, via next-themes
- **General feel:** Clean and minimal (Linear/Notion feel), lots of whitespace, clear hierarchy, warm but professional, confidence-building UI (progress indicators, checkmarks, clear completion states)
- **Avoid:** Cold/corporate, playful/startup-y, cluttered

## Logo

- **Icon:** Compass diamond (two-tone teal needle inside a circle with 4 tick marks)
- **Full logo:** Compass icon + "Sopia" wordmark
- **Files:** `public/logo-draft.svg` (full), `public/icon-draft.svg` (favicon/icon)
- **Colors:** Teal #2AA5A0 (light), #1D7A76 (dark), text #1a1a1a
